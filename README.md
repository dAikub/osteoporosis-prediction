Osteoporosis Prediction is about data classification for predicting osteoporosis. 
The goal was to compare the performance of different machine learning models through the evaluation process and Data Preparation, and then develop a web application that allows users to input data and see prediction results. 
I also created a research paper based on the project findings. that is the point of the project
